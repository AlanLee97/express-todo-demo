# Todo

练手项目，使用到的技术桟有：node.js、express、sequlize、mysql、vue、element-ui等



### 后端项目
技术桟
- node.js
- express.js
- sequlize
- mysql / mysql2

##### 安装依赖
```
cd todo-be
npm install
```

##### 启动项目
```
npm start
```



---



### 前端项目

技术桟：
- vue
- vue-cli
- element-ui
- axios
- moment
- sass

##### 安装依赖
```
cd todo-fe
npm install
```

##### 本地启动项目
```
npm run serve
```

##### 打包
```
npm run build
```
